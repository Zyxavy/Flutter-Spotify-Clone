# Flutter-Spotify-Clone
A UI Clone of Spotify Made with Flutter for the course: Mobile Programming 1


## Build 
```bash

# To build as Native Windows Application
flutter run -d windows

# To build to a web (Edge, Chrome)
flutter run -d web

# Build to a mobile
flutter run
```

### I recommend building using windows or web since it is faster and is mostly functional
