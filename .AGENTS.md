# Agent Instructions

## Project
Flutter Spotify Clone UI. UI-only, no backend, no audio.

## Stack
- Flutter + Riverpod + go_router + google_fonts (Nunito only)
- Run: `flutter run -d windows` or `flutter run -d edge`

## Rules
- Never hardcode hex colors → use AppColors only
- Never add new providers without noting it
- Font: Nunito (bodyM/bodyS). Montserrat (headings). NOT Circular.
- No http/dio/audioplayers — out of scope
- All shared widgets → lib/widgets/
- All mock data → lib/data/mock_data.dart

## Folder structure
lib/screens/ lib/widgets/ lib/data/ lib/theme/ lib/router/ lib/providers/

## Out of scope
Podcasts, social, real auth, real audio, notifications, downloads
